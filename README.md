# Ecommerce Order Processing System

A simple Java-based backend application for managing order processing in an e-commerce system.

---

## 🛠️ Requirements

| Tool      | Version   |
|----------|-----------|
| Java     | 17        |
| JDK      | 23        |
| Git      | Latest    |

---

## 🚀 Getting Started

Follow the steps below to set up and run the project on your local machine.

### 1. Clone the Repository

```bash
git clone https://github.com/gadilingappark/ecommerce-order-processing-system.git
cd ecommerce-order-processing-system


step 2: Switch to the correct branch

git checkout ecommerce_be

step 3: Verify Java Installation

Make sure Java version 17 is installed and configured, check with the below command:

java -version

step 4: Verify JDK Installation

Ensure JDK 23 is installed (required for compilation) use below command:

javac -version


step 5: run the project with main file with below command(and make sure you are in correct directory)

javac EcommerceApp.java
