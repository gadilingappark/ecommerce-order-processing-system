# Ecommerce Order Processing System

A simple Java-based backend application for managing order processing in an e-commerce system.

---

## 🛠️ Requirements

| Tool      | Version   |
|----------|-----------|
| Java     | 17        |
| JDK      | 23        |
| Git      | Latest    |

---

## 🚀 Getting Started

Follow the steps below to set up and run the project on your local machine.

### 1. Clone the Repository


git clone https://github.com/gadilingappark/ecommerce-order-processing-system.git
cd ecommerce-order-processing-system

step: 2. git checkout development

step 3. Verify Java Installation

Make sure Java version 17 is installed and configured with belwo command:

java -version

step: 4. Verify JDK Installation

Ensure JDK 23 is installed (required for compilation) with below command:

javac -version

step: 5. Run the project with the main file

Make sure you are inside the project root folder before executing below commands and below is the command to start the project

javac EcommerceApp.java
